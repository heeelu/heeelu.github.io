01

🔧BIOS / UEFI

  • › BIOS-Update: Auf der Hersteller-Website laden und via EZ Flash / M-Flash / Q-Flash flashen.
  • › XMP / EXPO aktivieren: XMP = Intel, EXPO = AMD — volle RAM-Geschwindigkeit freischalten.
  • › Resizable BAR aktivieren: Im BIOS unter PCIe-Einstellungen — kostenloser FPS-Boost.
  • › Lüfterkurve anpassen: Unter „Fan Control" / „Smart Fan" sanfte Kurve einstellen für leisen Betrieb.
02

📦Treiber & Software

  • › Windows-Updates: Einstellungen → Windows Update → Alles installieren. Danach auch Store-Apps aktualisieren.
  • › Mainboard-Treiber: Chipsatz, LAN, Audio direkt vom Hersteller — ASUS, MSI, Gigabyte.
  • › GPU-Treiber: NVIDIA App oder AMD Adrenalin — immer aktuell halten.
  • › Visual C++ All-in-One — verhindert „DLL not found"-Fehler in Spielen.
  • › .NET 3.5 aktivieren: Systemsteuerung → Windows-Features → Häkchen bei .NET Framework 3.5 setzen.
  • › Hersteller-Tools: Samsung Magician (SSD-Health), iCUE (RGB/Lüfter).
03

🖥️Windows-Optimierung

  • › Bildwiederholrate: Anzeigeeinstellungen → Erweiterte Einstellungen → maximale Hz wählen.
  • › NVIDIA / AMD auf Leistung: Im Treiber-Panel alle Einstellungen auf maximale Leistung setzen.
  • › Mausbeschleunigung aus: „Zeigergeschwindigkeit verbessern" deaktivieren — für präzises Aiming.
  • › Schnellstart deaktivieren: Sorgt für saubere Neustarts ohne veraltete Zustände.
  • › Autostart bereinigen: Strg + Shift + Esc → Tab „Autostart" → alles Unnötige deaktivieren.
  • › Energieoption: „Ausbalanciert" — optimal für Alltag & Gaming. Kein Grund für „Höchstleistung".
  • › GPU-Hardwarebeschleunigung: Einstellungen → Grafik → Hardwarebes. GPU-Planung AN + Spielmodus AN.
  • › Speicher-Integrität: Bei Bedarf deaktivieren für kleinen Leistungsgewinn.
04

🛠️Tools & Wartung

  • › X3D CPUs ⚠ WICHTIG — Xbox & Game Bar NICHT deinstallieren! Nötig für den 3D V-Cache Scheduler.
  • › Revo Uninstaller — entfernt Programme restlos inkl. Registry-Reste. Besser als Windows-Standard.
  • › Wöchentliche Systemwartung — PowerShell als Admin:
    PS>sfc /scannow
    PS>dism /online /cleanup-image /checkhealth
    PS>dism /online /cleanup-image /restorehealth
  • › Laufwerke optimieren: Windows-Suche → „Laufwerke defragmentieren" → SSDs: Optimieren, HDDs: Defragmentieren.
